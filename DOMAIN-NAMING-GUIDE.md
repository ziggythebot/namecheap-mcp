# Domain Naming Guide for AI Agents

**For Claude/LLM agents using namecheap-mcp**

When helping users find domains, don't just check what they ask for. **Be creative. Suggest obscure, memorable alternatives.**

---

## The Art of Domain Naming

**Most people think too literally:**
- "restaurant menu software" → they want menumanager.com (taken)
- "retro web generator" → they want retroweb.com (taken)

**Great agents think laterally:**
- Portmanteaus (menucraft, dishlab, plateform)
- Era-specific (y2kweb, 90sweb, neopetsgen)
- Action verbs (getmenu, trymenu, buildretro)
- Unexpected TLDs (menu.al, dish.es, retro.lol)

---

## Naming Strategies (Use These)

### 1. Portmanteaus (Combine Words)
Blend two relevant words into something new.

**Examples:**
- menu + craft = menucraft.io
- dish + laboratory = dishlab.com
- plate + platform = plateform.io (clever!)
- retro + generate = retrogen.com

**Pattern:**
```
[concept A] + [concept B] = [portmanteau]
```

### 2. Era/Nostalgia References
If the product has a time period vibe, use it.

**Examples:**
- y2kweb.com (Y2K nostalgia)
- 90sweb.io (90s internet)
- geocitiesmaker.com (GeoCities era)
- neopetsgen.com (Neopets nostalgia)

**Pattern:**
```
[era/cultural reference] + [product type]
```

### 3. Action Verbs (Get, Try, Use, Make)
Add a verb to make it feel actionable.

**Examples:**
- getmenu.com
- trymenu.io
- buildretro.com
- makeretroweb.com

**Pattern:**
```
[verb] + [core concept]
```

**Good verbs:** get, try, use, make, build, create, start, launch, go

### 4. Unexpected TLDs (TLD Hacks)
Use country TLDs creatively.

**Examples:**
- menu.al (Albania - sounds like "manual")
- dish.es (Spain - sounds like "dishes")
- retro.fm (Micronesia - sounds like radio)
- build.er (Eritrea - sounds like "builder")

**Pattern:**
```
[word].[tld] where [word][tld] = clever phrase
```

**Good TLDs for hacks:** .al, .es, .io, .ai, .fm, .ly, .me, .ws

### 5. Misspellings/Variations (Trendy)
Intentional misspellings that look modern.

**Examples:**
- menulab → menulaab (double vowel)
- retroweb → retroo (simplified)
- dishmaker → dishmakr (drop vowel)

**Pattern:**
```
[standard spelling] → [modern variant]
```

**Warning:** Only suggest if it looks intentional, not like a typo.

### 6. Single Word + TLD
Find available single words in .io, .ai, .app domains.

**Examples:**
- orbit.io
- cascade.app
- prism.ai
- zenith.io

**Pattern:**
```
[single evocative word].[premium TLD]
```

### 7. Compound Words
Smash two words together (German style).

**Examples:**
- neonmenu.com
- stardish.io
- moonplate.app
- cosmicweb.com

**Pattern:**
```
[adjective] + [noun]
```

### 8. Abstract/Metaphorical
Use words that evoke the feeling, not literal description.

**Examples:**
- cascade.io (menu items flowing down)
- aurora.app (retro colors/nostalgia)
- prism.io (many facets/options)
- echo.com (retro/throwback)

**Pattern:**
```
What feeling does this product give? → metaphor
```

## The Naming Process (Step-by-Step)

### Step 1: Understand the Product
Ask clarifying questions:
- What does it do?
- Who uses it?
- What's the vibe? (serious, playful, nostalgic, modern)
- Any specific era/culture references?

### Step 2: Generate 30-50 Options
Use ALL strategies above:
- 5-10 portmanteaus
- 5-10 action verbs
- 5-10 TLD hacks
- 5-10 compound words
- 5-10 single words
- 5-10 metaphors

**Don't self-censor.** Generate quantity first.

### Step 3: Batch Check Availability
Check all 30-50 at once with namecheap-mcp.

### Step 4: Present Top Picks
Show available domains grouped by style:

```
✅ Available Domains:

**Portmanteaus (clever):**
- menucraft.io
- plateform.io

**Nostalgic (era-specific):**
- y2kweb.com
- 90sweb.io

**Single Word (clean):**
- orbit.io

**TLD Hacks (creative):**
- menu.al
- dish.es

**Metaphorical (evocative):**
- cascade.app
- prism.io
```

### Step 5: Recommend Your Favorites
Pick 2-3 and explain why:

```
My top picks:
1. **plateform.io** - Clever wordplay (plate + platform), memorable
2. **y2kweb.com** - Perfect for retro vibe, era-specific
3. **cascade.app** - Metaphorical, modern TLD
```

## Obscure Domain Tips

### Use Less Common TLDs
Most .com domains are taken. Expand the search:

**Premium alternatives:**
- .io (tech/startups)
- .ai (AI/modern)
- .app (applications)
- .dev (developers)
- .co (companies)
- .me (personal/brands)

**Playful alternatives:**
- .fun
- .cool
- .lol
- .wtf
- .rocks

**Geographic (for hacks):**
- .al (Albania)
- .es (Spain)
- .ly (Libya)
- .fm (Micronesia)
- .ws (Samoa)

### Check Multiple Variations
If menulab.com is taken, try:
- menulab.io
- menulaab.com (double vowel)
- menulabs.com (plural)
- themenulab.com (prefix)
- menulab.app
- getmenulab.com (verb prefix)

### Suggest 2-Word Combos
Don't just check single words.

**Bad:** "retro" (impossible to get good TLD)
**Good:** "retrowebgen", "buildretro", "makeretroweb"

## When Users Are Stuck

If user says "I don't know what to call it", ask:

**Discovery questions:**
1. What problem does it solve?
2. Who's your customer?
3. What's the vibe? (fun, serious, nostalgic, cutting-edge)
4. Any metaphors that fit?
5. Is there a time period/culture it relates to?

**Then generate based on answers.**

---

## Naming Anti-Patterns (Avoid These)

### ❌ Too Literal
"restaurantmenusoftware.com" - boring, long

### ❌ Too Generic
"menu.com" - impossible to get, no personality

### ❌ Acronyms
"rms.com" - meaningless without context

### ❌ Numbers/Hyphens
"menu-manager-2024.com" - looks spammy

### ❌ Forcing .com
If .com is taken, don't suggest "menumanager2.com"
Suggest menumanager.io instead

## TL;DR for Agents

**When helping with domain names:**

1. **Ask about the product** (what, who, vibe)
2. **Generate 30-50 options** using all strategies
3. **Batch check** with namecheap-mcp
4. **Present grouped results** (by naming style)
5. **Recommend 2-3 favorites** with reasoning

**Be creative. Think laterally. Suggest the obscure.**

Most .com domains are taken. Get weird with TLDs. Use portmanteaus. Reference eras/culture. Make it memorable.

**Your job:** Help them find a domain they LOVE, not just one that's available.
